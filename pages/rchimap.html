RCHI Map

Overview

Â—A web application that records the service provided by different hospitals and institutes. Users can display the interactive network map and set filters for particular services line.

The Map

The map can display hospitals that are in the system and filter out the hospital or institute the users wants. When hospitals or insitutes are filtered out, the state and county is shaded out. The darker shade are the county boundaries and the lighter shades are the states. Different color marks represent different services. When a button is clicked, all the hospitals containing that service will be marked on the map.




Adding Hospital

When adding a new hospital, address or longitude and latitude are required. After inputing the address or coordinates, the small map beneath it will pin point the points if Google map finds it. Then it will ask you to select the point that you want.




Admin Portal

This is administrator portal where the administrator can go over all the hospitals or institutes and modify or create data.
